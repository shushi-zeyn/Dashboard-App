# Dashboard App

Application de visualisation de données construite avec Python et PyQt6.

## Fonctionnalités
- Importation de fichiers CSV et Excel.
- Génération automatique de graphiques.
- Choix des axes X et Y par l'utilisateur.
- Personnalisation du type de graphique.

## Installation
1. Créer un environnement virtuel : `python -m venv venv`
2. Activer l'environnement.
3. Installer les dépendances : `pip install -r requirements.txt`

## Lancement
Exécuter `src/main.py`.
