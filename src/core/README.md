# Core Logic

Ce dossier contient la logique métier complexe de l'application (génération des graphiques, algorithmes de traitement).

⚠️ **Note :** Le fichier `graph_manager.py` n'est pas inclus dans ce dépôt public pour des raisons de confidentialité / propriété intellectuelle.

Cette fonctionnalité a représenté un défi technique important à coder. Elle gère dynamiquement plusieurs types de graphiques (Lignes, Barres, Scatter, Pie) et doit s'adapter intelligemment aux différentes structures de données (CSV/Excel) et aux erreurs potentielles de l'utilisateur.
